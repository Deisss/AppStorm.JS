To use this example:

=== PYTHON ===
If python is not installed (note: tested with python 2.7):
	### WINDOWS ###
	Download and install: http://www.python.org/download/

	### UBUNTU/DEBIAN ###
	run "apt-get install python python-setuptools python-pip"
	=> jump to BOTTLE section directly

	### CENTOS/FEDORA/RPM... ###
	run "yum install python python-setuptools python-pip"
	=> jump to BOTTLE directly


=== SETUPTOOLS ===
Setup tools is a setup utility for Python:
	### WINDOWS ###
	download here: https://pypi.python.org/pypi/setuptools/0.9.8#installing-and-using-setuptools
	extract archive
	go into the folder where setup.py is located
	run "python setup.py install"


=== PIP ===
Pip is a package manager for python:
	### WINDOWS ###
	Download here: https://pypi.python.org/pypi/pip#downloads
	extract archive
	go into the folder where setup.py is located
	run "python setup.py install"



=== BOTTLE ===
Bottle is a tiny framework for creating REST app in python.

To install it:
	### ALL OS ###
	pip install bottle
